Plugins and Framework

  • Foundation by Zurb — responsive front-end framework
  • Reverie — WordPress framework based on ZURB's Foundation.
  • Flexslider — responsive jQuery slider plugin.
  • Modernizr — feature detection library for HTML5/CSS3
  • Normalize
  • Twitter's REST API by Abraham Williams
  • Custom Metaboxes and Fields for WordPress
  • Isotope
  • TGM Plugin Activation — PHP library to easily require plugins for WordPress themes.
